# Quizzard

A quiz app which is based on famous shows.

Live Link : https://quizzard-app.netlify.app/

## Tech Stack
 **Frontend**
 * React
 * Typescript
 * GlitchPop CSS (CSS component library made by me)
 
 **Backend**
 * NodeJs
 * Express
 * MongoDB
 
 ## Key features
 
 * JWT authorization
 * play quizes
 * view all scores on homepage
